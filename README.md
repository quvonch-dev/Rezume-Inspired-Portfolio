# Rezume Inspired Portfolio Clone

Bu loyiha Colorlib Rezume uslubidan ilhomlangan dark portfolio/resume landing page.

## Ichidagi fayllar

- `index.html` — asosiy HTML sahifa
- `css/style.css` — asosiy dizayn
- `css/media.css` — responsive media querylar
- `js/app.js` — mobile menu, filter, theme toggle, animation va contact form demo

## Ishga tushirish

1. Papkani VS Code bilan oching.
2. `index.html` faylini browserda oching.
3. O'zgartirishlarni HTML/CSS/JS fayllarida qiling.
